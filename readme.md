## Back-end em Node.js

Responsável por intermediar as requisições aos bancos de dados e fornecer retorno em JSON para o front-end web e mobile. Além de enviar e-mail's e tratar de toda regra de negócio.

#### Principais utilizações:

- Typescript
- TypeORM
- Redis
- MongoDB
- BCrypt
- Date-fns
- Class-transformer
- Nodemailer
- Multer
- Jest
